# CuCSS

"CustomCSS - injects custom CSS into pages."

## Dev

To test on Firefox, install the [web-ext](https://extensionworkshop.com/documentation/develop/getting-started-with-web-ext/)` cli util.

Run the project:
```
web-ext run --devtools
```

Validate the code:
```
web-ext lint
```
